How to start?

To install and save in your package.json dependencies, run:

npm i super-jss@latest

Add To your project

in your app.module, be sure to import SuperJssModule

imports: [SuperJssModule]

Basic usage

In any component.html add [sJss] to use the jss powers

Usage with breakpoints

This superpower allows the usage of breakpoints in the jss style to any property add the breakpoint object xs:string, md:string, sm:string, lg:string, xl:string

Usage with theme

This superpower allows the usage of a theme To do so, add the SuperJss Service in the ts constructor, and define a theme variable

Theme Override

override any theme definition

Theme Definition

This is the original theme, you can update any value
